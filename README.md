Bazaartisan - Find unique raw materials on this marketplace: https://bazaartisan.herokuapp.com/

Technical specs:

- Ruby v 2.5.1
- Gems keys : active storage AWS, Stripe, Mailchimp (Gibbon), font-awesome...
